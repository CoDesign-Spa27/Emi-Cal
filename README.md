
# Emi Calculator

This is a EMI Calculator Project
 

 
 
## Run Locally

Clone the project

```bash
  git clone https://github.com/CoDesign-Spa27/Emi-Cal.git
```

Go to the project directory

```bash
  cd Emi-Cal
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
